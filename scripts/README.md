# InsightEngine Scripts

Reusable scripts for the InsightEngine pipeline.

| Script | Purpose |
|--------|---------|
| `check_deps.py` | Check all required dependencies |
| `recalc.py` | Force Excel formula recalculation |

## Usage

All scripts accept CLI arguments — no hardcoded paths.

```bash
python3 scripts/check_deps.py
python3 scripts/recalc.py output.xlsx
```
